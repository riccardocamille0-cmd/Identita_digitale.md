# Identita_digitale.md
"Genera un file Markdown chiamato identita_digitale.md. Il file deve fungere da carta d'identità professionale e comportamentale per istruire un agente AI su chi sono e come deve interagire con me o per mio conto. Struttura il file con le seguenti sezioni:

Dati Anagrafici: Nome, cognome, data di nascita, luogo.

Profilo Professionale: Che lavoro faccio, competenze chiave e strumenti che uso.

Metodo di Lavoro: Come approccio i problemi, preferenze di gestione task, orari di produttività.

Stile di Comunicazione (Parlato): Tono di voce (formale/informale), intercalari, livello di tecnicismo.

Stile di Scrittura Email: Struttura tipica delle mail, saluti preferiti, brevità o prolissità.

Obiettivi dell'AI: Come l'AI deve assistermi (es. 'sii il mio critico', 'sii il mio assistente esecutivo').

Lascia i campi vuoti o usa dei placeholder tra parentesi quadre [ ] che io andrò a compilare."
